Event Description

The Cyber Threat Hunt, organized by OWASP KARE, is an intensive cybersecurity competition designed to challenge and enhance the skills of CSE, IT, and CS/IT students. This event simulates real-world cyber threat scenarios, pushing participants to apply theoretical knowledge in practical, high-pressure situations.

The competition is structured in two progressive rounds:

  • Round 1: Password Cracking, Phishing Identification, and Code Decryption challenges
  • Round 2: Vulnerable Machine Cracking tasks, testing participants' abilities to identify, exploit, and secure system vulnerabilities

Outcomes

  • Increased readiness for national-level events like "Pentathon" and international competitions such as “DEF CON CTF”, “SANS NetWars”, and “Global Cyberlympics”.
  • Enhanced practical cybersecurity skills through simulated real-world scenarios.
  • Hands-on experience with industry-standard tools like Wireshark, Nmap, Metasploit, Burp Suite, John the Ripper, and Ghidra.
  • Improved proficiency in network analysis, vulnerability scanning, penetration testing, web security, password cracking, and reverse engineering.
